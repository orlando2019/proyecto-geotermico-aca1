# Proyecto Geotérmico Nereidas - Análisis del Microentorno

Este proyecto es una aplicación web interactiva que presenta un análisis del microentorno del Proyecto Geotérmico Valle de Nereidas, desarrollado por CHEC Grupo EPM. La herramienta se divide en tres páginas principales que abordan diferentes aspectos estratégicos y de sostenibilidad.

## Características Principales

1. **Presentación del Caso**: Información general sobre el proyecto geotérmico y su contexto.
2. **Análisis Estratégico**:
   - Análisis FODA (Fortalezas, Oportunidades, Debilidades, Amenazas).
   - 5 Fuerzas de Porter con tooltips interactivos para cada fuerza.
3. **Cadena de Valor y Sostenibilidad**:
   - Cadena de valor del proyecto.
   - Benchmarking internacional (Colombia vs. Costa Rica).
   - Sostenibilidad ESG (Ambiental, Social y Gobernanza). Incluyendo descripciones termodinámicas como **entalpía**.

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica de las páginas.
- **CSS3**: Diseño responsivo, estilos personalizados, variables nativas, gradientes y tooltips interactivos.
- **JavaScript**: Lógica de navegación simple en la misma página (`showPage(n)`).

## Instrucciones de Uso

1. Clonar el repositorio o descargar el archivo `index.html`.
2. Abrir el archivo `index.html` en cualquier navegador web moderno.
3. Utilizar los botones de navegación en la parte inferior de cada página para avanzar o retroceder.
4. Pasar el cursor sobre las **5 Fuerzas de Porter** (Página 2) o la palabra **entalpía** (Página 3) para descubrir información adicional a través de tooltips interactivos.
